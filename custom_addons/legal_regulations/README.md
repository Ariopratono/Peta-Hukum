# Legal Regulations Module

Module untuk mengelola peraturan hukum dan perundang-undangan dengan lengkap.

## Fitur Utama:
- ✅ Manajemen peraturan hukum lengkap
- ✅ Informasi detail peraturan (nomor, bentuk, tahun, dll)
- ✅ Status dan validitas peraturan
- ✅ Pencarian dan filter advanced
- ✅ Klasifikasi bidang hukum
- ✅ Upload TXT/DOCX dengan auto-extract
- ✅ **BARU: Konsolidasi Peraturan** - Gabungkan UU Induk + UU Perubahan secara otomatis

## 🔗 Fitur Konsolidasi Peraturan

### Apa itu Konsolidasi?
Menggabungkan beberapa versi peraturan (UU Induk + UU Perubahan) menjadi satu dokumen terpadu dengan tracking perubahan yang detail.

**Contoh:** 
- UU 11/2008 (Induk) + UU 19/2016 (Perubahan) + UU 1/2024 (Perubahan Kedua)
- **Hasil:** Naskah terpadu dengan highlight pasal yang ditambah, diubah, atau dihapus

### Cara Menggunakan:
1. Pilih multiple regulations di tree view
2. Klik **Action** → **🔗 Buat Konsolidasi**
3. Pilih mode tampilan (Anotasi/Final/Riwayat/Perbandingan)
4. Klik **Generate & Simpan**

### Mode Tampilan:
- **Anotasi** ⭐ - Versi terbaru dengan highlight perubahan (RECOMMENDED)
- **Final** - Hanya versi terbaru, tanpa anotasi
- **Riwayat** - Semua versi per pasal
- **Perbandingan** - Side-by-side comparison
